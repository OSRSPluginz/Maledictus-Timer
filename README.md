# Maledictus Tracker

This plugin displays observations from player clients to track Revenant maledictus spawns across all worlds. It logs confirmed sightings, tracks upcoming spawn times, and provides icons to illustrate the timer.
## Features
- **Spawn Tracking:** Tracks Maledictus Spawns through NpcID and in-game chat messages.
- **Overlay:** Illustrates current world spawn timer and spawn prediction icon.
- **Panel:** Features a side panel with overlay toggle and world list with tracked times per world.

## Setup
1. Enable the plugin in the Plugin Hub.
2. Go to the plugin settings to configure overlay and panel settings.
